# Quick Start for Beginners: Diachronic Valency Corpus Super Platform

## 1. Setup (No Coding Needed)
1. Download or clone the project.
2. Open a terminal in the project folder.
3. Run:
   ```
   docker-compose up --build
   ```
4. Wait for all services to start (web, backend, database, agents).

## 2. Using the Platform
1. Open your browser to:
   ```
   http://localhost:3000
   ```
2. Login with:
   - Username: `admin`
   - Password: `changeme`
3. Upload your texts (Greek, English, Middle French, etc.).
4. Click “Validate” to approve real texts.
5. The platform will automatically:
   - Preprocess, annotate, and analyze texts with AI
   - Only process validated, non-quarantined texts
   - Run all agents 24/7 (no manual steps needed)
6. Browse, review, and export results from the web interface.

## 3. Safety & Review
- Password-protected and secure.
- Only validated texts are processed.
- All actions are logged for transparency.
- Easy review and correction workflow.

## 4. Troubleshooting
- If you see errors, check the logs in the terminal or web interface.
- For help, restart with:
  ```
  docker-compose restart
  ```
- If you need to reset your password, use the web interface or contact your admin.

## 5. In the Morning
- All results, logs, and flagged data will be ready for review.
- The platform is ready for real research and public/academic use!
