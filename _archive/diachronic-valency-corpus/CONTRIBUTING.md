# Contributing Guidelines

Thank you for your interest in contributing!

## How to Contribute
- Fork the repository and create a feature branch.
- Write clear, well-documented code and tests.
- Follow PEP8 (Python) and Prettier (JS) style guides.
- Add/modify tests for new features or bugfixes.
- Submit a pull request with a clear description.

## Code of Conduct
- Be respectful and constructive.
- Cite all academic and data sources.
- No scraping of copyrighted or non-open-access data.

## Reporting Issues
- Use GitHub Issues for bugs, feature requests, or questions.
